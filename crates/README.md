# crates

Rust workspace for the shared Haneulchi control plane.

- `hc-domain`: core enums and vocabulary.
- `hc-runtime`: PTY and session runtime placeholder.
- `hc-workflow`: workflow loading and validation placeholder.
- `hc-control-plane`: orchestration placeholder.
- `hc-storage`: SQLite, artifact, and Keychain boundary placeholder.
- `hc-api`: local Unix socket API placeholder.
- `hc-ffi`: Swift bridge placeholder.
- `hc-cli`: thin command-line client placeholder.
