# Integration Tests

Use this directory for multi-crate and app/runtime integration coverage.

- `rg01_pack_smoke.sh`: validates the RG-01 readiness acceptance pack artifacts.
- `rg02_pack_smoke.sh`: validates the RG-02 terminal quality acceptance pack artifacts.
- `rg03_pack_smoke.sh`: validates the RG-03 dry-run compatibility pack artifacts.
- `rg04_pack_smoke.sh`: validates the RG-04 workflow reload dry-run pack artifacts.
- `rg05_pack_smoke.sh`: validates the RG-05 review flow dry-run pack artifacts.
- `rg06_pack_smoke.sh`: validates the RG-06 snapshot parity dry-run pack artifacts.
- `rg07_pack_smoke.sh`: validates the RG-07 ops attention dry-run pack artifacts.
