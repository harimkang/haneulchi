# RG-01 Readiness Checklist

Use this checklist while validating the Welcome / Readiness Launcher flow.

1. Confirm first launch opens `WF-01` instead of dropping directly into Project Focus.
2. Confirm `shell`, `git`, `preset`, `keychain`, and `workflow` status rows are visible.
3. Confirm `Add Folder` selects a project and updates the selected-project summary.
4. Confirm `Continue with Generic Shell` remains available when preset readiness is degraded.
5. Confirm `Open Settings` reveals the Settings route and `Retry` refreshes readiness.
6. Confirm the first shell command result is captured in `logs/S-01-first-run.log`.
