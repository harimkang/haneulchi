# Preset Config

`default-presets.json` is the checked-in Sprint 2 preset registry baseline.

Each preset entry defines:

- `id`
- `title`
- `binary`
- `default_args`
- `capability_flags`
- `requires_shell_integration`

Sprint 2 supported presets:

- `claude`
- `codex`
- `gemini`
- `yazi`
- `lazygit`
